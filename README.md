## RankerBE

The [backend](https://github.com/berkaykeskino/RankerBE), [frontend](https://github.com/berkaykeskino/RankerUI), and database are deployed on Render, Vercel, and Aiven respectively.

The website can be accessed [here](https://ranker-ui-for-vercel.vercel.app/login).
On the first visit, initial requests may take longer to respond due to service cold starts. Once initialized, the application operates normally.

Ranker is designed as an application users can rank each other based on different features called rank types.
